# SPPS circuits

## Setup (optional)
Node v10 and higher is required. Install packages:
``` 
    npm install
```

Compile circuits:
```
    npm run compile
```

Download ceremony:
``` 
    cd ceremony
    wget https://hermez.s3-eu-west-1.amazonaws.com/powersOfTau28_hez_final_12.ptau
    cd ..
```

Create setup:
``` 
    npm run setup
```


## Tests

    npm run test
